Long-form documentation for ferrompi.

This module embeds the Markdown files from the docs/ directory directly into rustdoc via #[doc = include_str!(...)]. Each sub-module corresponds to one documentation artifact. The same content is available as plain Markdown on GitHub at docs/ and docs/adr/; see docs/README.md for a navigable index with relative-link support.
