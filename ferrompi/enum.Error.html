Expand description

Error types for MPI operations.

Variants§

§

AlreadyInitialized

MPI has already been initialized.

§

Mpi

MPI error with class, code, descriptive message, and optional operation name.

Fields

§class: MpiErrorClass

The error class (category of error).

§code: i32

The raw MPI error code.

§message: String

Human-readable error message from MPI_Error_string.

§operation: Option<&'static str>

The ferrompi operation that produced the error, if known.

§

InvalidBuffer

Invalid buffer provided (e.g., send/recv buffer size mismatch).

§

InvalidOp

Invalid reduction operation for the method being called (e.g., passing a non-MAXLOC/MINLOC op to allreduce_indexed).

§

NotSupported(String)

Operation not supported (e.g., MPI 4.0 persistent collectives on older MPI).

§

Internal(String)

Internal ferrompi error.

impl Error

Source

pub fn from_code(code: i32) -> Self

Create a structured error from an MPI error code.

Calls ferrompi_error_info to obtain the error class and human-readable message from the MPI runtime.

§Panics

Panics if called with MPI_SUCCESS (code 0).

Source

pub fn from_code_with_op(code: i32, operation: &'static str) -> Self

Create a structured error from an MPI error code with an operation name.

Behaves identically to Error::from_code but populates operation: Some(operation) on the resulting Error::Mpi variant, so callers can record which ferrompi wrapper produced the error.

§Panics

Panics if called with MPI_SUCCESS (code 0), just like from_code.

Source

pub fn check(code: i32) -> Result<()>

Check an MPI return code, returning Ok(()) for success.

Returns Err(Error::Mpi { .. }) for non-zero codes.

Source

pub fn check_with_op(code: i32, operation: &'static str) -> Result<()>

Check an MPI return code with an operation name, returning Ok(()) for success.

Returns Err(Error::Mpi { operation: Some(operation), .. }) for non-zero codes.
