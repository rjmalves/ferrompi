Expand description

Trait for types eligible for use with MPI_Compare_and_swap.

This is a sealed trait — it cannot be implemented outside this crate. Per MPI 4.1 section 12.5.4, the predefined types guaranteed to support MPI_Compare_and_swap are the integer and byte types: i32, i64, u32, u64, and u8.

Floating-point types (f32, f64) are intentionally excluded. The MPI standard does not require implementations to support floating-point CAS. Accepting them would introduce silent portability bugs across MPI implementations.

This trait is used as a bound on Win::compare_and_swap to ensure the method only exists for windows whose element type is CAS-eligible.
